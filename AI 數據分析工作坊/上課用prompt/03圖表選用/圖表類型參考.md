# 圖表類型選擇速查指南 (Chart Chooser)

這份指南旨在幫助您根據要傳達的「數據故事」核心目的，快速選擇最合適的圖表類型。在選擇圖表前，先問自己一個問題：**「我想讓觀眾從數據中看到什麼？」**

通常，數據視覺化的目的可以歸結為以下四大類：

1.  **比較 (Comparison)**：比較不同類別之間的數值大小。
2.  **構成 (Composition)**：展示整體中各部分的佔比。
3.  **分佈 (Distribution)**：觀察數據的分布情況、找出異常值。
4.  **關聯 (Relationship)**：探討不同變量之間的關係。

---

### 1. 比較 (Comparison)

用於比較不同項目或同一項目在不同時間點的數值大小。

| 圖表類型 | 最佳使用時機 |
| :--- | :--- |
| **柱狀圖 (Bar Chart)** | 當分類少於 10 個時，用於比較不同分類的數值大小。最經典、最易讀的比較圖表。 |
| **條形圖 (Horizontal Bar Chart)** | 當分類標籤名稱過長，或者分類超過 10 個時使用。 |
| **折線圖 (Line Chart)** | 用於展示數據在一個**連續時間序列**上的變化趨勢。 |
| **雷達圖 (Radar Chart)** | 用於在多個（通常 > 4個）量化指標上，對不同對象進行綜合表現的比較。 |
| **分組柱狀圖 (Grouped Bar Chart)** | 在多個分類下，再對其次級分類進行數值比較。 |
| **子彈圖 (Bullet Chart)** | 將一個數值與其目標值和表現區間（如：差、良、優）進行比較，常用於儀表板。 |

---

### 2. 構成 (Composition)

用於展示一個整體中，不同部分所佔的比例。

| 圖表類型 | 最佳使用時機 |
| :--- | :--- |
| **餅圖 (Pie Chart)** | **謹慎使用！** 僅在分類少於 5 個且想強調某部分與整體的佔比關係時使用。 |
| **環圈圖 (Donut Chart)** | 餅圖的變體，中間的空白區域可以用來顯示總數或核心指標。 |
| **堆疊柱狀圖 (Stacked Bar Chart)** | 比較不同分類的總量，同時展示每個分類內部各組成部分的佔比。 |
| **百分比堆疊柱狀圖 (100% Stacked Bar Chart)** | 比較不同分類中，各組成部分的相對比例，而非絕對數值。 |
| **樹狀圖 (Treemap)** | 當分類具有層級關係，且想在有限空間內展示大量分類的佔比時使用。 |
| **瀑布圖 (Waterfall Chart)** | 展示一個初始值經過一系列正向或負向的變化後，如何得到最終值的過程。 |

---

### 3. 分佈 (Distribution)

用於觀察單個變量或多個變量的數據分布情況，以了解其集中趨勢、離散程度和異常值。

| 圖表類型 | 最佳使用時機 |
| :--- | :--- |
| **直方圖 (Histogram)** | 展示單一連續變量的數值分布頻率。 |
| **箱形圖 (Box Plot)** | 展示一組數據的五數概括（最小值、第一四分位數、中位數、第三四分位數、最大值），非常適合用來比較多組數據的分布並識別異常值。 |
| **密度圖 (Density Plot)** | 直方圖的平滑版本，能更清晰地看出分布的形狀。 |
| **散點圖 (Scatter Plot)** | 觀察兩個連續變量之間的分布關係。 |
| **熱力圖 (Heatmap)** | 使用顏色深淺來展示二維矩陣中數值的分布情況。 |

---

### 4. 關聯 (Relationship)

用於探討兩個或多個變量之間是否存在某种關係。

| 圖表類型 | 最佳使用時機 |
| :--- | :--- |
| **散點圖 (Scatter Plot)** | 判斷兩個連續變量之間是否存在線性、非線性或聚集關係的最佳選擇。 |
| **氣泡圖 (Bubble Chart)** | 散點圖的變體，用氣泡的大小來表示第三個變量的數值。 |
| **組合圖 (Combo Chart)** | 將柱狀圖和折線圖結合，用於展示兩個不同量綱的變量之間的關係（例如：銷量與利潤率）。 |
| **桑基圖 (Sankey Diagram)** | 用於視覺化流程或系統中能量、資源或金錢的流動與轉換關係。 |
