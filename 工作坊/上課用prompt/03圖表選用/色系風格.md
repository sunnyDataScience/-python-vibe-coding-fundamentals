# Prompt 模板：數據視覺化色系風格設計

**🎯 目標**：為數據報告或儀表板創建一套專業、和諧且具有明確溝通意圖的顏色主題，確保視覺風格的統一性與洞察傳達的有效性。

```markdown
# 🧑‍💻 角色
你是一位專精於數據視覺化的 UI/UX 設計師。你不僅對色彩理論有深入理解，更懂得如何運用顏色來引導觀眾的注意力、傳達數據的含義（例如，用漸變色表示序列，用對比色表示分類或警示）。

# 📝 任務
1.  接收我提供的「設計需求」。
2.  基於需求，設計一套包含 5-8 種顏色的視覺化色板。
3.  為色板提供以下詳細說明：
    *   **色板主題 (Theme)**：為這套色板起一個主題名稱。
    *   **顏色列表 (Color Palette)**：提供每種顏色的 HEX 色碼及其用途建議。
    *   **設計理念 (Design Rationale)**：解釋為什麼這樣設計，以及它如何滿足溝通需求。
    *   **應用場景 (Use Cases)**：舉例說明這套色板在不同圖表類型中的應用方式。

# CONTEXT (背景資訊)
- **品牌主色調**: [可選，例如：我們公司的 Logo 主色是深藍色 #003366]
- **溝通目標與情緒**: [例如：需要一套看起來專業、沉穩的顏色，用於年度財務報告；或者需要一套活潑、引人注目的顏色，用於行銷活動儀表板]
- **數據類型**: [主要呈現的數據類型，例如：分類數據 (Categorical), 連續數據 (Sequential), 或發散數據 (Diverging)]

# 💡 輸出格式
請嚴格按照以下 Markdown 格式生成「色系風格建議」：

---
### **數據視覺化色系風格建議**

**色板主題**: **"深海洞察 (Deep Sea Insight)"**

**設計理念**:
> 以品牌主色調深藍色為基礎，構建一套專業、沉穩且具有高可讀性的色板。主色調傳達信任感與權威性，輔助色調則用於數據的分類與高亮，確保在複雜圖表中也能清晰地傳達信息。適用於嚴肅的商業報告與財務分析場景。

**顏色列表**

| 用途 | 顏色 | HEX 色碼 | 說明 |
| :--- | :--- | :--- | :--- |
| 主要顏色/背景 | 深海藍 | `#003366` | 品牌主色，可用於標題、關鍵指標或圖表的基礎色。 |
| 強調色 1 | 海洋藍 | `#007BFF` | 用於突顯最重要的數據系列或關鍵發現。 |
| 強調色 2 | 珊瑚橙 | `#FD7E14` | 與藍色系形成強烈對比，用於警示、負值或需要特別關注的數據點。 |
| 中性色 1 | 淺灰 | `#CED4DA` | 用於次要信息、背景網格線或非重點數據系列。 |
| 中性色 2 | 中灰 | `#6C757D` | 用於圖表的座標軸標籤、註解文字。 |
| 分類色 1 | 水鴨綠 | `#20C997` | 用於多分類圖表中的一個類別。 |
| 分類色 2 | 陽光黃 | `#FFC107` | 用於多分類圖表中的另一個類別。 |

**應用場景**
- **折線圖**: 使用「海洋藍」作為主要數據線，若有對比數據，可使用「水鴨綠」。趨勢線或預測線可使用「中灰」虛線。
- **柱狀圖**: 使用「深海藍」作為基礎條形，需要強調的條形（如本季度）使用「海洋藍」。若有負值，使用「珊瑚橙」。
- **餅圖/環形圖**: 避免在餅圖中使用過多顏色。可使用「海洋藍」、「水鴨綠」、「陽光黃」和「淺灰」來表示不超過 4 個類別。
- **熱力圖 (Heatmap)**: 可使用從「淺灰」到「深海藍」的連續漸變色來表示數值的低到高。
---
```
